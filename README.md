## 配置要求


### limbo (边缘服务器/挂机服)

服务器平台
- 云服务器(VPS): ✔
- 面板服: ✔

资源要求
- CPU: 1
- 内存: 2GB
- 宽带: >3M
- 储存: 3GB


### atlas (大地图)

服务器平台
- 云服务器(VPS): ✔
- 面板服: ✔

资源要求
- CPU: 8 (cpuz单核 600+)
- 内存: 16GB
- 宽带: 20M
- 储存: 40GB


### mainCity (主城)

服务器平台
- 云服务器(VPS): ✔
- 面板服: ✔

资源要求
- CPU: 8
- 内存: 8GB
- 宽带: 20M
- 储存: 5GB (地图大小+3GB)


### Velocity
服务器平台
- 云服务器(VPS): ✔
- 面板服: ✔ (不推荐)

资源要求
- CPU: 2
- 内存: 2GB
- 宽带: 30M
- 储存: 3GB



### mysql 数据库
